# Tobacco Usage data from Missouri residents (BRFSS).

A sub-dataset containing tobacco usage information from the years 2023
and 2024.

## Usage

``` r
tobacco_use
```

## Format

A data frame with 14,528 rows and 10 variables:

- SEQNO:

  key of the dataset

- SMOKER3:

  general smoking status

- SMOKE100:

  smoked at least 100 cigarettes

- SMOKE100_IMP1:

  1st imputation of SMOKE100

- SMOKE100_IMP2:

  2nd imputation of SMOKE100

- SMOKE100_IMP3:

  3rd imputation of SMOKE100

- ECIGNOW:

  e-cigarettes usage

- ECIGNOW_IMP1:

  1st imputation of ECIGNOW

- ECIGNOW_IMP2:

  2nd imputation of ECIGNOW

- ECIGNOW_IMP3:

  3rd imputation of ECIGNOW
