# Cognitive Decline data from Missouri residents (BRFSS).

A sub-dataset containing cognitive decline information from the years
2023 and 2024.

## Usage

``` r
cognitive_decline
```

## Format

A data frame with 14,528 rows and 5 variables:

- SEQNO:

  key of the dataset

- CIMEMLO1:

  difficulties with thinking or memory

- CIMEMLO1_IMP1:

  1st imputation of CIMEMLO1

- CIMEMLO1_IMP2:

  2nd imputation of CIMEMLO1

- CIMEMLO1_IMP3:

  3rd imputation of CIMEMLO1
