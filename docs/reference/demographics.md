# Demographic data from Missouri residents (BRFSS).

A sub-dataset containing demographics information from the years 2023
and 2024.

## Usage

``` r
demographics
```

## Format

A data frame with 14,528 rows and 18 variables:

- SEQNO:

  key of the dataset

- MARITAL:

  marital status

- EDUCA:

  education level

- VETERAN3:

  veteran status

- EMPLOY1:

  employment status

- CHILDREN:

  number of children in household

- INCOME3:

  income level

- PREGNANT:

  pregnancy status1

- WEIGHT2:

  reported weight in pounds

- CRACE1:

  child non-hispanic race including multiracial

- CAGEG:

  child age

- AGE80:

  age

- GENHLTH:

  general health

- MENTHLTH:

  mental health

- CHCOCNC1:

  melanoma or other types of cancer

- ALCDAY4:

  days in the past 30 had alcohol

- MARJSMOK:

  marijuana usage

- ASTHMS1:

  asthma status
