Cleans Missouri data from the 2023 and 2024 BRFSS surveys. Specifically,
the cleaning focuses on the Tobacco Use and Cognitive Decline modules.
This package imputes related columns, providing three different imputed
versions of the column, as well as the original unimputed column data.
Other data present in the package includes basic survery information,
such as date, and demographics. The data included in this package is
split into four datasets--missouri_brfss, demographics, tobacco_use, and
cognitive_decline.
